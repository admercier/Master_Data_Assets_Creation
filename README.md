# Master_Data_Assets_Creation
Created three separate master data assets (SQL) for our data warehouse that combined data from both Clarity and Caboodle databases for robust solution for end users and reporting team. 

1. Patient Master
2. Visit Master
3. Encounter Master

The three queries were developed and then tables were created in our data warehouse. Then custom data flows were created in Power BI for not only the reporting team to utilize but alos end users who build in Power BI. The solution allowed for a more seamless consumption of the data (elimination of IDs and mapping table joins) as well as more data returned due to joining Caboodle and Clarity databases.
